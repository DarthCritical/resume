# Jared Pethel — Resume

Personal resume and portfolio site built with HTML, CSS, and JavaScript.

## Live site

Published with GitHub Pages: **https://darthcritical.github.io/resume/**

## About

Static single-page site with:

- About section and headshot
- Portfolio project highlights
- Cover letter for career transition into IT
- Contact links (LinkedIn, GitHub, email)

## Local development

Open `index.html` in a browser, or serve the folder with any static file server.

## Git commit identity

When committing from the command line, use the GitHub-provided noreply email so commits link to your profile and pass email privacy checks:

```bash
git config user.name "DarthCritical"
git config user.email "182289669+DarthCritical@users.noreply.github.com"
```

Find your noreply address under **GitHub → Settings → Emails** when "Keep my email addresses private" is enabled.
